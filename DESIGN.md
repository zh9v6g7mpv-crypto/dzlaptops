# DZLaptops — Design System

A bold, modern, dark-first aesthetic: electric **blue → violet** gradients,
soft glassmorphism, and an aurora glow. Built to look premium and cool.

## Look & feel
- **Dark-first** (light theme available via toggle). Near-black `#05060A` canvas.
- **Aurora backdrop** — blurred blue / violet / cyan glow behind the hero and page headers.
- **Glass cards** — translucent surfaces with backdrop blur so the aurora glows through.
- **Gradient accents** — headlines, buttons, icon tiles and the CTA band use the
  `#3B82F6 → #8B5CF6` blue-violet gradient.

## Color
| Token | Value |
|-------|-------|
| Blue | `#3B82F6` |
| Violet | `#8B5CF6` |
| Cyan (aurora) | `#22D3EE` |
| Gradient | `linear-gradient(120deg, #3B82F6, #8B5CF6)` |
| Dark canvas | `#05060A` |
| Light canvas | `#FBFBFD` |

## Type
System-first for a native, premium feel that needs no download:
`-apple-system / SF Pro` → **Inter** fallback (loaded from Google Fonts),
with **Noto Sans Arabic** for RTL. Large, tight-tracked display headings.

## Signature
Hero **laptop** rendered in SVG with a blue-violet gradient screen, glass
reflection and a soft glow halo — floating over the aurora.

## Quality floor
Responsive to mobile, visible keyboard focus, `prefers-reduced-motion`
respected, light/dark parity, FR/EN/AR with full RTL, and content that still
renders with JavaScript disabled.
